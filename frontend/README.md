# Frontend (React)

The frontend provides the user interface for Synapse Guard.

## Responsibilities
- Login form + JWT storage
- Dashboard UI
- Transaction list + CRUD operations
- Displaying fraud scores (color-coded)
- Communicating with all backend microservices

## Technologies
- React
- TypeScript
- Vite
- Axios or Fetch API

## Folder Structure
- `src/` – Main React files
- `public/` – Static assets
